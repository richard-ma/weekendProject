# 内存

## 需求
* DDR3 DDR4
* 4G 8G
* ECC

## 内存信息
| 内存规格 | 二手市场参考价 | 行情备注 |
| :--- | :--- | :--- |
| DDR3 4G (普通) | 约 20 - 50 元 | 价格已回落至炒作前水平，普通条价格较低 |
| DDR3 4G (ECC) | 约 15 - 35 元 | 服务器拆机条居多，需主板支持ECC，价格通常低于普通条 |
| DDR3 8G (普通) | 约 60 - 100 元 | 相比高峰期（曾达200元）大幅下跌 |
| DDR3 8G (ECC) | 约 40 - 70 元 | 常见于服务器平台（如X79/X58等），性价比高，但兼容性要求高 |
| DDR4 4G (普通) | 约 30 - 50 元 | 适合老旧平台升级，价格相对低廉 |
| DDR4 4G (ECC) | 约 25 - 45 元 | 较少见，多为服务器入门级拆机条，需确认主板支持 |
| DDR4 8G (普通) | 约 150 - 185 元 | 近期有清仓抛售现象，价格处于下行区间 |
| DDR4 8G (ECC) | 约 120 - 160 元 | 服务器专用（如RDIMM），普通家用主板通常无法点亮，价格略低于普通条 |
| DDR4 16G (普通) | 约 280 - 350 元 | 主流容量，价格随市场波动较大，近期呈下降趋势 |
| DDR4 16G (ECC) | 约 220 - 300 元 | 服务器大容量拆机条，适合组建工作站或NAS，性价比极高 |
